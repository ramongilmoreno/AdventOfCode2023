Advent of code 2023
===================

Web site and login

  https://adventofcode.com/2023

  Use Google account to log in.


Quick start guide

  For each challenge there is a directory "<day>-<part>"
  
  For exmple, for Day 1 there are two parts 1 and 2, thus these directories:
  
    01-1/
    01-2/
 
  In each directory, there is a ./run.sh command to solve the challenge:

    $ ./run.sh
